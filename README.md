# Arduino UNO Q Projects

This repository contains working projects developed alongside the book:

**Arduino UNO Q — Step-by-Step System Control Using Arduino and Python**

## Method Used

All projects follow a two-phase approach:

Phase 1 — Arduino IDE (MCU control)  
Phase 2 — App Lab (Python → Bridge → MCU)

## Projects

### Two-Phase: LED Control
Basic proof of system control using the built-in LED.

### Two-Phase: Motor Control
Control of a DC motor using the TA6586 driver and Python.

## Structure

Each project contains:

- `sketch/` → Arduino MCU code  
- `python/` → Linux (Python) control code  
- `README.md` → instructions and expected results  

## Purpose

These projects are not examples—they are working systems.

Each step is tested and produces a measurable result.
